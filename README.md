# 🎓 Student Management System

A **Python-based Student Management System** built using **Object-Oriented Programming (OOP)** principles. This console application allows users to efficiently manage student records by performing operations such as adding, searching, deleting, and saving student data in CSV format.

The project demonstrates the practical implementation of OOP concepts, file handling, and data management while maintaining a clean and user-friendly interface.

---

## 📌 Features

- ➕ Add new student records
- 🔍 Search students by name
- 🗑️ Delete student records
- 💾 Save student data to a CSV file
- 👤 Student introduction feature
- 🎂 Birthday (Age Increment)
- 🎓 Honour Student Identification
- 📊 Track total number of students
- 🏫 University information using Class Variables
- 📁 CSV File Handling

---

## 🛠️ Technologies Used

- **Python 3**
- **CSV Module**
- **Object-Oriented Programming (OOP)**
